# Fallout 4 Hacker Tool

A "handy" tool to hack terminals in Fallout 4. I guess can also use it in Fallout 76.

Fallout 4 用のハッキングヘルパーです。たぶん76でもつかえます。

## Build and Use

1. Run `yarn install`
2. Run `yarn build`
3. open `dist/index.html`

## Run development server

1. Run `yarn install`
2. Run `yarn start`
